# О проекте

**Epistola** — кроссплатформенный мессенджер на Flutter + Firebase, вдохновлённый Telegram.

Проект начинался как небольшой мессенджер для ~40 пользователей, но архитектура сразу строится с расчётом на дальнейшее развитие и масштабирование.

Основные цели:

- личные чаты;
- групповые чаты;
- роли и модерация;
- настройки групп;
- Spaces (пространства);
- мультимедиа;
- уведомления;
- видеозвонки;
- Android, iOS и Web.

## Роли

Поддерживаются:

- owner
- admin
- moderator
- member
- guest

Owner пока зарезервирован под будущее расширение.

Создатель группы получает роль:

```
admin
```

---

## Модерация

Роли отделены от наказаний.

Статусы:

### normal

Обычный пользователь.

### muted

Можно читать, нельзя писать.

Поддерживаются:

- временный mute;
- постоянный mute.

### banned

Пользователь не может пользоваться группой.

Поддерживаются:

- временный ban;
- постоянный ban.

---

## Автоматическая очистка наказаний

Реализована:

```
clearExpiredMemberStatus()
```

Просроченные mute и ban автоматически снимаются.

# Безопасное управление группами (v0.5.1)

Реализовано:

### Передача прав администратора

Администратор может передать права другому участнику.

После передачи сам становится участником.

---

### Защита последнего администратора

Последний администратор не может:

- выйти из группы;
- понизить себя;
- оставить группу без управления.

---

### Роспуск группы

Реализован механизм:

```
isDissolved = true
```

После роспуска:

- группа скрывается у всех участников;
- писать в неё нельзя;
- история пока сохраняется;
- позже возможно физическое удаление.
